# fish-ahtarsk
Создать специализированный вебпортал для рыбаков и любителей отдыха на воде, который объединит:  Онлайн-трансляции с ключевых рыболовных мест (лиманы, море, реки).  Прогноз клёва с учётом погоды, давления, фазы луны.  Интерактивную карту рыболовных мест с отметками от пользователей.  Информацию о базах отдыха, прокате лодок и снаряжения, магазинах.
